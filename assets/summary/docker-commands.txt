* Sec03
- run docker (to create the dbs)
    - docker compose up -d
- see the volumes that created from docker
    - docker volume list
- delete docker volume
    - docker volume rm volume_name
- delete docker container
    - docker compose down
- build service
    - docker compose build search-svc
- build all iamges in docker-compose.yml
    - docker compose build
* Sec07
- Stop container on current dir if there is a docker-compose.yml
    - docker-compose down  
- Remove all containers
    - docker rm -fv $(docker ps -aq)
- get all active ports
    - sudo lsof -PiTCP -sTCP:LISTEN
- List who's using the port
    - sudo lsof -i -P -n | grep <port number>
- kill process
    - kill <process id>
