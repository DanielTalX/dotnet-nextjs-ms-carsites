* sec02
- list and info for the diffrent dotnet sdk that have installed in the computer
    dotnet --info
- list of commands that can create with dotnet sdk:
    dotnet new list
- create a solution (folder name is the solution name):
    - cd Carsties
    - dotnet new sln
- create a webapi project with dotnet v8:
    dotnet new webapi -o src/AuctionService --use-controllers
- add the webapi project to Carsties solution:
    dotnet sln add src/AuctionService/
- debug/run AuctionService:
    - cd src/AuctionService/
    - dotnet watch
- db migration:
    - using dotnet-ef tool: to check if dotnet-ef tool installed globaly in the computer by: dotnet tool list -g
        if not, install it by: dotnet tool install dotnet-ef -g, or update it by: dotnet tool update dotnet-ef -g
    - [AuctionService] add migration: dotnet ef migrations add "InitialCreate" -o Data/Migrations
    - [AuctionService] drop the db: dotnet ef database drop
* sec04
- create a shared lib:
    - [Carsties] dotnet new classlib -o src/Contracts
    - [Carsties] dotnet sln add src/Contracts
- add a reference to the shared library from a service
    - [AuctionService] dotnet add reference ../../src/Contracts
